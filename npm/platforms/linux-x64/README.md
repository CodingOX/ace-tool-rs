# @alistar.max/ace-ctx-linux-x64

Platform-specific binary for [ace-ctx](https://github.com/CodingOX/ace-ctx) (Linux x64).

## Overview

This package contains the pre-built binary for Linux x64. It is automatically installed as an optional dependency when you run `npx @alistar.max/ace-ctx` on Linux x64 systems.

**Do not install this package directly.** Instead, use:

```bash
npx @alistar.max/ace-ctx --base-url <API_URL> --token <AUTH_TOKEN>
```

## Supported Platforms

- Linux x64/amd64
