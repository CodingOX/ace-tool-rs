# @alistar.max/ace-ctx-darwin-universal

Platform-specific binary for [ace-ctx](https://github.com/CodingOX/ace-ctx) (macOS Universal - x64 and arm64).

## Overview

This package contains the pre-built binary for macOS (Universal). It is automatically installed as an optional dependency when you run `npx @alistar.max/ace-ctx` on macOS.

**Do not install this package directly.** Instead, use:

```bash
npx @alistar.max/ace-ctx --base-url <API_URL> --token <AUTH_TOKEN>
```

## Supported Platforms

- macOS x64 (Intel)
- macOS arm64 (Apple Silicon)
