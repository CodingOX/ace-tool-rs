# @alistar.max/ace-ctx-win32-arm64

Platform-specific binary for [ace-ctx](https://github.com/CodingOX/ace-ctx) (Windows ARM64).

## Overview

This package contains the pre-built binary for Windows ARM64. It is automatically installed as an optional dependency when you run `npx @alistar.max/ace-ctx` on Windows ARM64 systems.

**Do not install this package directly.** Instead, use:

```bash
npx @alistar.max/ace-ctx --base-url <API_URL> --token <AUTH_TOKEN>
```

## Supported Platforms

- Windows arm64
